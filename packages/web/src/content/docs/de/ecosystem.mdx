Eine Sammlung von Community-Projekten, die auf 2M_CODE basieren.

:::note
Möchten Sie Ihr 2M_CODE-bezogenes Projekt zu dieser Liste hinzufügen? Senden Sie einen PR.
:::

Sie können sich auch [awesome-2M_CODE](https://github.com/awesome-2M_CODE/awesome-2M_CODE) und [2M_CODE.cafe](https://2M_CODE.cafe) ansehen, eine Community, die das Ökosystem und die Community zusammenfasst.
