:::

คุณยังสามารถลองดู [awesome-2M_CODE](https://github.com/awesome-2M_CODE/awesome-2M_CODE) และ [2M_CODE.cafe](https://2M_CODE.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
