Una colección de proyectos comunitarios construidos en 2M_CODE.

:::note
¿Quieres agregar tu proyecto relacionado con 2M_CODE a esta lista? Envíe un PR.

También puedes consultar [awesome-2M_CODE](https://github.com/awesome-2M_CODE/awesome-2M_CODE) y [2M_CODE.cafe](https://2M_CODE.cafe), una comunidad que agrega el ecosistema y la comunidad.
